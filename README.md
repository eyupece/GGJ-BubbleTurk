# GGJ-BubbleTurk: Bubble Man's Adventure 🎮

## Proje Hakkında
Bu proje, Global Game Jam 2024 etkinliği için geliştirilmiş bir 2D platform oyunudur. Oyun, Godot Engine 4.3 kullanılarak geliştirilmiştir.
Global Game Jam Link: https://globalgamejam.org/games/2025/popocalypse-1

## 🎯 Oyun Özellikleri
- 2D platform oyunu
- Steampunk temalı silah sistemi
- Hikaye odaklı oynanış
- Ana menü ve duraklatma menüsü
- Kredi ekranı
- Özel font kullanımı (Bubblez-Regular)

## 🛠️ Teknik Detaylar
- **Oyun Motoru:** Godot Engine 4.3
- **Programlama Dili:** GDScript
- **Proje Yapısı:**
  - `animations/`: Oyun animasyonları
  - `models/`: 3D modeller
  - `objects/`: Oyun nesneleri
  - `scripts/`: GDScript kodları
  - `shaders/`: Shader dosyaları
  - `textures/`: Doku dosyaları

## 🎮 Kontroller
- **Hareket:** WASD veya Ok tuşları
- **Silah Değiştirme:** Mouse 4
- **Ateş Etme:** Sol Mouse Tuşu
- **Gece Değiştirme:** N tuşu
- **Menü İptal:** ESC tuşu

## 🚀 Kurulum
1. Projeyi klonlayın
2. Godot Engine 4.3'ü indirin ve yükleyin
3. Projeyi Godot Engine ile açın
4. `main_Menu.tscn` sahnesini çalıştırın
